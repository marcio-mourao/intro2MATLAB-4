Exercises

Exercise #1 - Strings, Cell Arrays and Structures

1.1 Create a variable A whose value is the string ‘hello’.

1.2 Check its length.

1.3 Concatenate A with your own name and assign the outcome to a new variable.

1.4 Create a cell array B containing the strings: ‘hello’ ‘goodbye’ ‘hello’ ‘hello goodbye’ ‘goodbye’ ‘hello’ ‘goodhellow’. Each index in your cell array should contain one of these strings. Note, you can create an empty cell array by setting it to an empty cell, e.g. B = {}; and you can add a string to the end of a cell array by e.g. B{end+1} = 'hello';

1.5 Create a structure C with two fields named 'field1' and 'field2'. Assign them with any string you want and a 2x2 matrix of random values. Now, change the matrix from the structure to [1 2;3 4]. Make sure the changes are properly done in the structure.

2.1 Sort rows of table T by column Height, starting with the highest value

2.2 Obtain mean and std Height of males and females in the table T

2.3 Obtain scaled Weight values and assign the outcome to a new variable

2.4 Confirm that the values obtained in the previous point are indeed scaled properly

2.5 Create a new table A including only male smokers

else c. n = -10 m = ?

3.3 Create a function that calculates the factorial of a number. The function displays a warning message (not performing any additional operations) if the input number is negative.

3.4 Write a brief script to evaluate the following:

h(T) = T - 10 when 0 < T < 100

= 0.45 T + 900 when T > 100

Exercise #4 - Loop Constructs

a. Add up the values of the elements (Check with sum.)

b. Compute the running sum (for element j, the running sum is the sum of the

elements from 1 to j, inclusive. Check with cumsum.) Example: [1 2 3] -> [1 3 6]

c. Compute the sine of the given x-values (should be a vector)

4.2 Create an M-by-N array of random numbers (use rand). Move through the array, element by element, and set any value that is less than 0.2 to 0 and any value that is greater than (or equal to) 0.2 to 1.

4.3 Do the same as in 4.2 using boolean indexes.
